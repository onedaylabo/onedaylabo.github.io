# 実装プラン - 今の仕事、辞めるべき？診断

「仕事を辞めたい気がするけど決断できない人」をターゲットに、感情ではなく整理された視点を提供する診断アプリを開発します。

## 1. デザインシステム (CSS)
- **カラーパレット**: 
  - 背景: ダークネイビー (`#0f172a`)
  - カード: 透過ネイビー (`rgba(30, 41, 59, 0.7)`)
  - アクセント: ソフトブルー (`#38bdf8`) / セージグリーン (`#10b981`)
  - テキスト: オフホワイト (`#f8fafc`) / グレー (`#94a3b8`)
- **フォント**: `Outfit`, `Inter`, 游ゴシック体
- **エフェクト**: グラスモフィズム、マイクロインタラクション（ボタンの浮き上がり）、フェードアニメーション

## 2. アプリケーション構造
- `index.html`: 全体の構造（トップ、質問、結果の各セクション）
- `style.css`: プレミアムデザインの定義
- `script.js`: 診断ロジック、データ管理、UI制御
- `assets/`: 画像素材

## 3. 実装ステップ
1. **基盤作成**: 共通のスタイルと基本レイアウトの構築。
2. **質問データ定義**: 30問の質問をJSON形式で定義。
3. **診断エンジン**: 1問ずつ表示し、回答を収集するロジックの実装。
4. **スコアリング & 結果生成**: スコア計算と、結果レベル（4段階）に応じたメッセージ生成。
5. **ビジュアル強化**: 背景画像の生成とアニメーションの追加。

## 4. 特徴的な機能
- **進捗インジケーター**: 全30問のどこにいるかを視覚化。
- **メンタル・レッドフラグ**: 心身の健康スコアが一定値を超えた場合、合計点に関わらず優先的に警告。
- **モバイル最適化**: 夜にベッドでスマホで見ているユーザーに合わせたデザイン。
